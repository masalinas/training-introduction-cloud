Docker Datacenter

Docker offers a Container-as-a-Service (CaaS) solution, the Docker Datacenter, which is built on top of the Universal Control Plane (UCP) and Docker Trusted Registry (DTR). Docker Datacenter is an enterprise container management and deployment service hosted on-premise behind a firewall or in the cloud.

With Docker Datacenter we can build an enterprise-class CaaS platform on-premises, as it is built on top of Docker Swarm and integrates well with Docker tools, and the Docker Registry. In addition, it provides LDAP and AD integration, monitoring, logging, and plugin support for network and storage.
