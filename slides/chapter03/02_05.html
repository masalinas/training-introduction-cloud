Cloud Foundry BOSH

Cloud Foundry (CF) utilizes two runtimes to separately manage and run applications and containers. These are the Cloud Foundry Application Runtime and Cloud Foundry Container Runtime. 

The tool that manages both runtimes for CF is Cloud Foundry BOSH. Although it works in the background and developers may not be exposed to the runtime management tool, BOSH is a cloud-agnostic open source tool for release engineering, deployment, and lifecycle management of complex distributed systems.
