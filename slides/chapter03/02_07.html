Cloud Foundry Application Runtime (CFCR)

Cloud Foundry Container Runtime (CFCR) offers Cloud Foundry users the alternative to deploy cloud-native, developer-built, pre-packaged applications in containers. The CFCR platform manage containers on a Kubernetes cluster, which in turn is managed by Cloud Foundry BOSH. Kubernetes is a container orchestrator that operates in conjunction with a container runtime and automates deployment, scaling, and management of containerized applications. CF BOSH adds high availability, scaling, VM healing, and upgrades support to the Kubernetes cluster.
